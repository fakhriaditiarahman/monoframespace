## 2024-04-16 - Prevent Hook Rules Violations inside Arrays and Props
**Learning:** Found critical React anti-patterns that directly violate the Rules of Hooks: using `useTransform` directly inline inside style props (`style={{ opacity: useTransform(...) }}`), and calling hooks inside an array `.map()` loop. In a framework like Next.js executing in a highly animated page, this can severely impact performance by causing memory leaks, garbage collection spikes, and hook order instability.
**Action:** Always extract inline hook calls to top-level function declarations and always extract loops into dedicated child components to securely use hooks within mapped data structures. Ensure strict adherence to the Rules of Hooks for smooth runtime performance and stable memory utilization.
